# lattice-hamiltonian

**Ising and Potts models on custom lattices** — simulate statistical mechanics models with configurable Hamiltonians, annealing schedules, and lattice topologies.

## What This Gives You

- **Ising model** — spin-½ systems with ferromagnetic/antiferromagnetic couplings
- **Potts model** — q-state generalization with configurable interaction strengths
- **Custom lattices** — define arbitrary lattice topologies (2D grid, triangular, hexagonal, random)
- **Annealing** — simulated annealing with configurable temperature schedules
- **Observables** — energy, magnetization, specific heat, correlation functions

## How It Fits

Statistical mechanics engine for the SuperInstance mathematical research. Connects to `constraint-hamiltonian` (constraint formulation) and `lattice-climate` (climate modeling applications).
